# solar-system-pygame
## Controls
-   UP/DOWN for Emulation Speed Control
-   ESC for Pause/Unpause
	-	q to Quit
	-	c or ESC again to Continue
-	LEFT CLICK to Add an orbiting Planet to the current position
-	RIGHT CLICK to Hide/Show Orbit circles
